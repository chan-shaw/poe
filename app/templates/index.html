Poe API 测试工具

测试您的 Poe API Token 并检查模型可用性

API 使用说明

本服务提供与 OpenAI API 兼容的接口，可以使用任何支持 OpenAI API 的客户端进行调用。

1. 获取模型列表

curl -X GET "http://localhost:35555/v1/models" \
-H "Authorization: Bearer YOUR_POE_API_KEY"

2. 发送聊天请求

curl -X POST "http://localhost:35555/v1/chat/completions" \
-H "Content-Type: application/json" \
-H "Authorization: Bearer YOUR_POE_API_KEY" \
-d '{
  "model": "MODEL_ID",
  "messages": [
    {"role": "user", "content": "你好，请介绍一下自己"}
  ]
}'

3. 流式响应

curl -X POST "http://localhost:35555/v1/chat/completions" \
-H "Content-Type: application/json" \
-H "Authorization: Bearer YOUR_POE_API_KEY" \
-d '{
  "model": "MODEL_ID",
  "messages": [
    {"role": "user", "content": "你好，请介绍一下自己"}
  ],
  "stream": true
}'

使用说明

  1. 输入您的 Poe API Token
  2. 选择语言代码（默认为中文）
  3. 点击"获取模型列表"按钮
  4. 对每个模型点击"测试"按钮检查可用性
  5. 查看测试结果和响应内容
提示：测试结果仅表示模型是否可以正常响应，不代表模型质量。
